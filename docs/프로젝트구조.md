## 검색 페이지 (`/search`)

`app/search/page.tsx`

### 검색 폼

#### 4가지 필터

- **날짜**: date input, YYYY-MM-DD 형식, 필수
- **서비스**: 멀티 셀렉트 드롭다운 (본식스냅, 서브스냅, 영상촬영, 아이폰스냅, 돌스냅, 애견스냅, 야외촬영, 스튜디오촬영)
- **지역**: 단일 선택 드롭다운
- **가격**: 단일 선택 드롭다운

### 검색 결과

#### 결과 요약 카드

매칭된 작가 수 표시

#### 작가 카드 그리드

각 카드에 포함되는 정보:
- 대표 이미지
- 작가명
- 서비스 종류
- 지역
- 가격
- 평점
- 성향 키워드
- 즐겨찾기(하트) 버튼
- 포트폴리오/영상 링크

### 사이드바

- **최근 본 작가**: 접이식 목록, localStorage 저장
- **즐겨찾기 드롭다운**: localStorage 저장

### 데이터 흐름

```
검색 폼 입력 → GET /api/search?date=&region=&price=&service=
→ Airtable artists + closed_dates 테이블 조회
→ 인메모리 필터링 (지역, 서비스, 가격, 촬영 불가 날짜 제외)
→ 랜덤 셔플 후 반환
→ 카드 그리드 렌더링
```

### 상태 관리

- **sessionStorage**: 검색 조건 및 스크롤 위치 보존 (작가 상세 → 뒤로가기 시 복원)
- **localStorage**: 최근 본 작가, 즐겨찾기 목록

### 네비게이션

- 작가 카드 클릭 → `/artists/{id}`

## 사진 업로드 (`/artist-upload`)

### 기능

- 최대 40장 이미지 업로드 (장당 10MB 제한)
- 대표 사진 지정
- 기존 이미지 미리보기
- 개별 파일 삭제

### 업로드 흐름

```
이미지 파일 선택
→ POST /api/cloudinary/sign (업로드 서명 발급)
→ Cloudinary에 직접 업로드 (클라이언트 → Cloudinary)
→ PATCH /api/artists/{id}/portfolio-images (URL 목록 저장)
→ PATCH /api/artists/{id}/representative-image (대표 사진 저장)
```

### Cloudinary 업로드 구조

- 폴더: `daypic/artists/representative`
- 서명 방식: SHA1(`folder={folder}&timestamp={timestamp}{apiSecret}`)
- 클라이언트 사이드 업로드 (서버는 서명만 발급)

### 네비게이션

### 작가 관리

#### `GET /api/artists/[id]`

작가 상세 조회. `recXXX` 형식이면 직접 조회, 아니면 `artist_id`/`kakao_id`/`user_id`로 검색.

#### `PATCH /api/artists/[id]`

작가 프로필 수정. 요청에 포함된 필드만 업데이트. 배열/문자열 자동 판별.

#### `POST /api/artists/register`

신규 작가 등록. 모든 필수 필드 검증 후 Airtable에 레코드 생성. `검색노출: true` 자동 설정.

#### `GET /api/artists/by-email`

이메일로 작가 조회. `{이메일}="{email}"` 수식으로 검색.

#### `PATCH /api/artists/[id]/portfolio-images`

포트폴리오 이미지 URL 목록 저장. 최대 40장. 줄바꿈으로 구분하여 long text 필드에 저장.

#### `PATCH /api/artists/[id]/representative-image`

대표 사진 URL 저장. Airtable attachment 형식 `[{ url: "..." }]`으로 변환.

#### `PATCH /api/artists/[id]/video-portfolio`

영상 포트폴리오 저장. `video_link_1~4`, `video_thumb_1~4`, `video_style_tags` 업데이트.

---

### 일정 관리

#### `GET /api/artist-closed`

로그인 작가의 촬영 불가 날짜 목록 조회.

#### `POST /api/artist-closed`

촬영 불가 날짜 등록. 중복 시 `duplicated: true` 반환 (에러 아님).

#### `DELETE /api/artist-closed?date={date}`

촬영 불가 날짜 해제.

---

### 인증

#### `GET & POST /api/auth/[...nextauth]`

NextAuth 핸들러. 카카오 OAuth 프로바이더. JWT 전략.

**콜백 흐름**:
1. `signIn`: 카카오 인증 정보 로깅
2. `jwt`: kakaoId 추출 → Airtable `user` 테이블에서 email로 조회 → userId, artistId 토큰에 추가
3. `session`: 토큰 값을 세션 객체에 전달

#### `GET /api/me`

현재 세션 정보 반환. JWT에서 kakaoId 추출 → Airtable artists 테이블에서 `kakao_id`로 조회 → 작가 여부 판별.

---

### 파일 업로드

#### `POST /api/cloudinary/sign`

Cloudinary 업로드 서명 발급. 클라이언트에서 직접 Cloudinary로 업로드할 때 사용.

**폴더**: `daypic/artists/representative`

**서명**: `SHA1(folder={folder}&timestamp={timestamp}{apiSecret})`

## Airtable 테이블 구조

### artists 테이블

주요 필드 (한국어명 / 영어 폴백):
- `업체명` / `name` — 작가명
- `이메일` / `email` — 작가 이메일
- `촬영서비스` / `service` — 서비스 목록
- `촬영지역` / `region` — 지역 목록
- `촬영비용` / `price` — 가격 구간
- `성향키워드` / `keywords` — 스타일 키워드
- `대표사진` / `image` — 대표 이미지 (attachment)
- `포트폴리오` / `portfolio` — 포트폴리오 URL
- `openchat_url` — 카카오 오픈채팅 링크
- `portfolio_images` — 포트폴리오 이미지 URL 목록
- `video_link_1~4` — 영상 링크
- `video_thumb_1~4` — 영상 썸네일 (attachment)
- `video_style_tags` — 영상 스타일 태그
- `artist_type` — 작가 유형
- `kakao_id` — 카카오 ID (인증 연동)
- `user_id` — 사용자 ID
- `검색노출` — 검색 노출 여부 (boolean)

### closed_dates 테이블

- `날짜` / `date` — 촬영 불가 날짜 (YYYY-MM-DD)
- `artist_id` — 작가 ID
- `작가이메일` / `artist_email` — 작가 이메일
- `owner_user_id` — 등록한 사용자 ID
- `date_key` — `{email}_{date}` 형식의 복합 키

### user 테이블

- `email` — 사용자 이메일
- `user_id` — 사용자 ID

## 환경변수

| 변수 | 용도 |
|------|------|
| `AIRTABLE_TOKEN` | Airtable API 인증 토큰 |
| `AIRTABLE_BASE_ID` | Airtable Base ID |
| `AIRTABLE_ARTISTS_TABLE` | 작가 테이블명 (기본: "artists") |

| `NEXTAUTH_SECRET` | NextAuth JWT 서명 시크릿 |
| `KAKAO_CLIENT_ID` | 카카오 OAuth 클라이언트 ID |
| `KAKAO_CLIENT_SECRET` | 카카오 OAuth 클라이언트 시크릿 |
| `CLOUDINARY_CLOUD_NAME` | Cloudinary 클라우드 이름 |
| `CLOUDINARY_API_KEY` | Cloudinary API 키 |
| `CLOUDINARY_API_SECRET` | Cloudinary API 시크릿 |

## 상태 관리 패턴

| 저장소 | 용도 | 사용 페이지 |
|--------|------|-------------|
| Airtable | 작가 프로필, 불가 날짜, 사용자 계정 | 모든 API 라우트 |
| localStorage | 최근 본 작가, 즐겨찾기, 체크리스트 상태 | `/search`, `/checklist` |
| sessionStorage | 검색 조건, 스크롤 위치 | `/search` |
| NextAuth JWT | 인증 토큰 (kakaoId, userId, artistId) | 인증 필요 페이지 전체 |
